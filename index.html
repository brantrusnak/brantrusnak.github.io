CSS Keyframe Delay Generator

I developed this tool to assist in the generation of CSS keyframes with animation timings, delays/pauses, and states. CSS animations can only be delayed at the start, so to delay you have to break the keyframe into 2 steps, the first being the animation time and the second being the delay time. This tool does the math and typing required in calculating seconds to percentages in keyframes. I used this to generate the "I love to" animation in the about card above.

  • HTML5
  • CSS3
  • JavaScript
